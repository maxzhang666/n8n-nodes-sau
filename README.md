# n8n-nodes-sau

小红书支持二维码登录（扫描登录不用担心账号密码泄露），笔记搜索，笔记发布等功能.
## 功能说明：
* ✅ 笔记发布
* ✅ 笔记搜索（支持筛选条件参考下方说明）
* ✅ 笔记点赞
* ✅ 笔记收藏
* ✅ 笔记评论
* ✅ 获取笔记基本信息
* ✅ 获取笔记互动数据(点赞数、收藏数、评论数)
* ✅ 获取笔记图片
* ✅ 获取笔记视频
* ✅ 搜索用户
* ✅ 获取用户笔记
* ✅ 支持代理ip隔离
* ❎ 一机多账号登录
